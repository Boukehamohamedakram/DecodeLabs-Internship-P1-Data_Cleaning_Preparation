# DecodeLabs Internship Project P1

This project contains the cleaned dataset and supporting structure for the DecodeLabs Data Cleaning & Preparation track.

## Project structure

- `config/` — environment and configuration templates
- `data/raw/` — original raw dataset files
- `data/processed/` — cleaned and transformed datasets
- `data/outputs/` — analysis outputs and reports
- `docs/` — project documentation

## Getting started

1. Copy `config/.env.example` to `config/.env`
2. Update `EDA_INPUT_DATA_PATH` and other environment variables
3. Place raw data in `data/raw/`
4. Save analysis results to `data/outputs/`
